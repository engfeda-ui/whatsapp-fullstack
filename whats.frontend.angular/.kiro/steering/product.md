# Product Overview

## WhatsApp Business Frontend

A modern, production-ready Angular 19 frontend for WhatsApp Business management with enterprise-grade UI components and real-time capabilities.

### Core Purpose

Provides a comprehensive web interface for managing WhatsApp Business devices, sending messages, tracking delivery, and integrating AI-powered features for content generation and chat assistance.

### Key Features

- **Device Management**: CRUD operations for WhatsApp Business devices with QR code pairing
- **Message Operations**: Single and bulk message sending with delivery tracking
- **Real-Time Updates**: WebSocket integration for live device status and message notifications
- **AI Integration**: AI-powered chat assistance, content generation, and reply suggestions
- **Authentication**: JWT-based auth with token encryption and refresh token support
- **Subscription Management**: Plans display, usage tracking, and payment management
- **Performance**: HTTP caching, lazy loading, and optimized bundle size

### Target Users

- WhatsApp Business account holders
- Customer support teams
- Marketing teams managing bulk messaging
- Developers integrating WhatsApp Business APIs

### Supported Backends

- ASP.NET Core (recommended)
- NestJS

### Deployment Targets

- Azure Static Web Apps
- Netlify
- Vercel
- Docker/Docker Compose
- Any static hosting with HTTPS
